# Quirizz Frontend
A frontend application using NextJs for RISTEK Recruitment that focuses on the Quirizz application.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Axios

## Getting Started
Quick guide on how to get started with this project.

## Prerequisites
- Node.js (>= 14.x)
- npm or yarn
- Git

## Installation

### Clone the repository
```bash
git clone https://github.com/ezarsurahman/quirizz-frontend.git
```
### Move into the directory
```bash
cd quirizz-frontend
```
### Install all dependencies
Using npm:
```
npm install
```
Using yarn:
```
yarn install
```
### Run the application
Using npm:
```
npm run dev
```
Using yarn:
```
yarn dev
```
